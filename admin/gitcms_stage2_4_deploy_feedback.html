Scope it to the site repo only, Contents: read & write. It is sent only to api.github.com.
How to create the token
  1. GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token.
  2. Resource owner: your account. Repository access: Only select repositories → the site repo.
  3. Permissions → Repository permissions → Contents: Read and write. Nothing else.
  4. Pick a short expiry. Paste it above — it never touches any repo.
